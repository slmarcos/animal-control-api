# Buscar todos os animais cadastrados

## Sucesso
---
  - [ ] Recebe uma requisição **GET** no endpoint **/api/animals**.
  - [ ] Busca todos os animais
  - [ ] Retorna o código de status **200** e uma lista com os dados dos animais

## Falha
---
  - [ ] Retorna erro **500** se ocorrer uma falha interna no servidor
