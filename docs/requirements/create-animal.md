# Criar um novo animal

## Sucesso
---
  - [X] Recebe uma requisição **POST** com os dados do animal no endpoint **/api/animal**
  - [X] Valida os campos campos obrigatórios do body da requisição
  - [X] Cria um novo animal
  - [X] Retorna os dados do animal com o código de status **200**

## Falha
---
  - [X] Retorna erro **500** se ocorrer uma falha interna no servidor
  - [X] Retorna erro **400** com uma mensagem de erro se faltar algum campo do body
