# Criar um novo animal

## Sucesso
---
  - [ ] Recebe uma requisição **POST** com os dados do animal no endpoint **/api/animal**
  - [ ] Valida os campos campos obrigatórios do body da requisição
  - [ ] Cria um novo animal
  - [ ] Retorna os dados do animal com o código de status **200**

## Falha
---
  - [ ] Retorna erro **500** se ocorrer uma falha interna no servidor
  - [ ] Retorna erro **400** com uma mensagem de erro se faltar algum campo do body
